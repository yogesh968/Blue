# Person 3 - Hospital & Emergency Services

- Hospital management system
- Bed booking functionality
- Ambulance services
- Emergency services portal

## Setup Instructions
1. Copy these files to the main project structure
2. Ensure database has Hospital, BedBooking, and Ambulance models
3. Test hospital registration and bed booking
4. Set up emergency services workflow

- Frontend: Hospital pages, emergency portal, service listings

- Integrates with payment system from Person 4
- Emergency services accessible to all user types
